> Ya! Noviyanto menggunakan CMS yang ramah pengguna dan memberikan pelatihan. Update paket, foto, harga, dan blog — tanpa coding.

**Q: Bagaimana jika saya tidak puas dengan hasilnya?**

**Q: Ada biaya rutin setelah website jadi?**

**Q: Bisa dicicil pembayarannya?**

## CATATAN IMPLEMENTASI CRO

> Panduan teknis developer & designer untuk mengoptimalkan konversi halaman ini.

| Elemen | Spesifikasi |
|---|---|
| **URL** | `noviyanto.com/jasa-website-tour-travel` |
| **Struktur** | Single-page (1 file HTML) dengan anchor navigation |
| **Warna Primary** | `#1A5276` (Biru gelap) |
| **Warna Accent/CTA** | `#E67E22` (Oranye) |
| **Warna Background Gelap** | `#0D1B2A` |
| **Font Heading** | Poppins Bold / Montserrat Bold |
| **Font Body** | Inter / Open Sans, 16–18px |
| **Navbar behaviour** | Sticky fixed top, scroll spy aktif |
| **CTA Placement** | Hero → setelah Fitur (mid-page CTA) → setelah Harga → Final CTA |
| **WhatsApp Float** | Button sticky `wa.me/6285156724581` di kanan bawah layar (selalu visible) |
| **CTA Button size** | Min height 52px, padding horizontal 32px (mobile-friendly) |
| **Loading target** | < 3 detik. Kompres semua gambar, lazy load galeri |
| **Hero image** | Foto destinasi wisata Indonesia resolusi tinggi + overlay `rgba(0,0,0,0.65)` |
| **Scroll animation** | Fade-in + slide-up ringan (AOS.js atau CSS) untuk setiap section |
| **Mobile breakpoint** | 768px (tablet), 480px (mobile). Mobile-first CSS |
| **Back to main site** | Logo di navbar = link ke `noviyanto.com` |
| **Analytics** | Pasang Google Tag Manager + goal tracking per CTA button |
